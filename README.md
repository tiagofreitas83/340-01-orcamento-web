# Casa Seasons A36-A37 — Orçamento (web)

Visualizador publicado do orçamento da casa Seasons (Stemmer 340-01, lotes A36/A37, Xangri-lá/RS — proprietários Avelino e Betty, casa de veraneio).

**Acesso protegido por senha.** Apenas o conteúdo encriptado client-side (AES-256-GCM via PBKDF2) está neste repo.

Fonte: repo privado [`tiagofreitas83/340-01-orcamento`](https://github.com/tiagofreitas83/340-01-orcamento).

Para atualizar: rodar `./publish.sh` no repo privado (regenera `orcamento.html`, encripta com staticrypt e dá push aqui).
